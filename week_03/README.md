# Неделя 3 - Условные операторы и математические вычисления

## Обзор недели

Третья неделя посвящена изучению условных операторов, математических вычислений и работе с различными типами данных в Go. Студенты изучают основы принятия решений в программах и выполнение математических операций.

## Задания

### [Задание 1: Обработка трех чисел](./task_01/)

**Описание:** Работа с условными операторами для обработки трех попарно различных действительных чисел.

**Ключевые темы:**
- Условные операторы `if-else`
- Работа с вещественными числами (`float64`)
- Математические вычисления
- Проверка условий и логические операции

**Файлы:**
- [main.go](./task_01/main.go) - исходный код
- [README.md](./task_01/README.md) - подробное описание задания
- [go.mod](./task_01/go.mod) - модуль Go

**Запуск:**
```bash
cd week_03/task_01
go run main.go
```

## Цели обучения

После завершения недели 3 студенты должны уметь:

1. **Использовать условные операторы** для принятия решений в программах
2. **Работать с вещественными числами** и выполнять математические вычисления
3. **Обрабатывать пользовательский ввод** с проверкой корректности
4. **Применять математические функции** из стандартной библиотеки Go
5. **Структурировать код** с использованием функций для решения задач

## Технические навыки

- Условные операторы (`if`, `else if`, `else`)
- Switch-case конструкции
- Математические операции и функции
- Работа с типами данных `float64` и `int`
- Обработка ошибок ввода
- Структурирование кода с функциями
