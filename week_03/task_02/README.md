# Week 03, Task 02: Case12 - Вычисление элементов окружности

## Описание задания

Элементы окружности пронумерованы следующим образом:
1. **Радиус** (R)
2. **Диаметр** (D = 2 * R)
3. **Длина окружности** (L = 2 * π * R)
4. **Площадь круга** (S = π * R²)

**Задача:** По номеру одного из элементов (1, 2, 3 или 4) и его значению вычислить и вывести значения остальных элементов данной окружности в том же порядке (Радиус, Диаметр, Длина окружности, Площадь).

**Ограничение:** Использовать значение π = 3.14

## Требования

- Программа должна принимать на вход номер элемента (1-4) и его значение
- Вычислять все остальные элементы окружности
- Выводить результат в заданном порядке
- Обрабатывать ошибки ввода
- Проверять корректность входных данных

## Формулы

- **Диаметр:** D = 2 * R
- **Длина окружности:** L = 2 * π * R
- **Площадь круга:** S = π * R²
- **Радиус из диаметра:** R = D / 2
- **Радиус из длины:** R = L / (2 * π)
- **Радиус из площади:** R = √(S / π)

## Структура программы

- `Circle` - структура для хранения всех элементов окружности
- `calculateFromRadius()` - вычисление всех элементов по радиусу
- `calculateFromDiameter()` - вычисление всех элементов по диаметру
- `calculateFromCircumference()` - вычисление всех элементов по длине окружности
- `calculateFromArea()` - вычисление всех элементов по площади
- `printCircle()` - вывод всех элементов в заданном порядке
