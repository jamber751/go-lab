# Week 03, Task 03: Решение квадратного уравнения

## Описание задания

Даны действительные числа a, b, c (a ≠ 0). Выяснить, имеет ли уравнение ax² + bx + c = 0 действительные корни. Если действительные корни имеются, то найти их. В противном случае ответом должно служить сообщение, что действительных корней нет.

## Требования

- Программа должна принимать на вход коэффициенты a, b, c
- Проверять, что a ≠ 0
- Вычислять дискриминант D = b² - 4ac
- Определять количество действительных корней:
  - Если D < 0: корней нет
  - Если D = 0: один корень x = -b/(2a)
  - Если D > 0: два корня x₁ = (-b + √D)/(2a), x₂ = (-b - √D)/(2a)


## Алгоритм

1. Ввод коэффициентов a, b, c
2. Проверка условия a ≠ 0
3. Вычисление дискриминанта D = b² - 4ac
4. Анализ дискриминанта:
   - D < 0 → "Действительных корней нет"
   - D = 0 → один корень x = -b/(2a)
   - D > 0 → два корня по формуле
5. Вывод результата
