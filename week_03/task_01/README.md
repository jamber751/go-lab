# Задание 1 - Обработка трех чисел

## Описание задачи

Даны три попарно различных действительных числа x, y, z. Необходимо выполнить следующие действия:

1. **Если сумма трех чисел меньше единицы:**
   - Найти наименьшее из трех чисел
   - Заменить его полусуммой двух других чисел

2. **Если сумма трех чисел больше или равна единице:**
   - Найти меньшее из чисел x и y
   - Заменить его полусуммой двух оставшихся значений

## Требования

- Программа должна работать с действительными числами
- Необходима проверка на попарную различность чисел
- Программа должна включать тестовые случаи
- Должна быть возможность интерактивного ввода

## Примеры работы

### Пример 1: Сумма < 1
```
Исходные числа: x=0.10, y=0.20, z=0.30
Сумма: 0.60
Результат: x=0.25, y=0.20, z=0.30
```
*Наименьшее число x=0.10 заменено на полусумму y и z: (0.20+0.30)/2 = 0.25*

### Пример 2: Сумма >= 1
```
Исходные числа: x=0.50, y=0.60, z=0.70
Сумма: 1.80
Результат: x=0.50, y=0.60, z=0.70
```
*Меньшее из x и y (x=0.50) заменено на полусумму y и z: (0.60+0.70)/2 = 0.65*

## Структура программы

- `main()` - основная функция с тестовыми случаями и интерактивным вводом
- `processNumbers(x, y, z float64)` - функция обработки чисел согласно алгоритму
- Проверка на попарную различность чисел
- Логика выбора наименьшего числа
- Вычисление полусуммы для замены
