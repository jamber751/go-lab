# Неделя 6 - Массивы и многомерные структуры данных

## Обзор недели

Шестая неделя посвящена изучению массивов и многомерных структур данных в Go. Студенты учатся работать с одномерными и двумерными массивами, выполнять сортировку, поиск элементов и анализировать данные.

## Задания

### [Задание 1: Работа с одномерным массивом](./task_01/)

**Описание:** Программа для работы с одномерным массивом из 12 элементов: ввод, сортировка по убыванию, поиск экстремумов и вычисление их суммы.

**Ключевые темы:**
- Одномерные массивы (`[]float64`)
- Алгоритмы сортировки
- Поиск максимального и минимального элементов
- Интерактивный ввод данных
- Обработка ошибок ввода

**Функциональность:**
- Ввод 12 действительных чисел
- Сортировка массива по убыванию
- Поиск максимального и минимального элементов
- Вычисление суммы экстремумов
- Дополнительно: вычисление среднего арифметического

**Файлы:**
- [main.go](./task_01/main.go) - исходный код
- [README.md](./task_01/README.md) - подробное описание задания
- [go.mod](./task_01/go.mod) - модуль Go

**Запуск:**
```bash
cd week_06/task_01
go run main.go
```

### [Задание 2: Сортировка массива с подсчетом перестановок](./task_02/)

**Описание:** Программа для сортировки массива по убыванию с подсчетом количества перестановок элементов.

**Ключевые темы:**
- Алгоритм пузырьковой сортировки (Bubble Sort)
- Подсчет операций сортировки
- Анализ эффективности алгоритмов
- Два режима ввода: ручной и случайная генерация
- Статистика производительности

**Алгоритм:**
- Сравнение соседних элементов
- Обмен элементов при необходимости
- Подсчет каждой перестановки
- Оптимизация: остановка при отсутствии перестановок

**Файлы:**
- [main.go](./task_02/main.go) - исходный код
- [README.md](./task_02/README.md) - подробное описание задания
- [go.mod](./task_02/go.mod) - модуль Go

**Запуск:**
```bash
cd week_06/task_02
go run main.go
```

## Цели обучения

После завершения недели 6 студенты должны уметь:

1. **Работать с одномерными массивами** - создание, заполнение, обработка
2. **Применять алгоритмы сортировки** и понимать их сложность
3. **Использовать двумерные массивы** для решения задач с матрицами
4. **Анализировать производительность алгоритмов** и подсчитывать операции
5. **Генерировать случайные данные** для тестирования программ

## Технические навыки

### Работа с массивами
- Создание и инициализация массивов
- Индексация и доступ к элементам
- Копирование массивов
- Передача массивов в функции

### Алгоритмы сортировки
- Пузырьковая сортировка (Bubble Sort)
- Встроенная сортировка Go (`sort.Slice`)
- Анализ временной сложности O(n²)
- Оптимизация алгоритмов сортировки

### Многомерные структуры
- Создание двумерных массивов
- Обход матриц с помощью вложенных циклов
- Работа с произвольными размерами матриц
- Генерация случайных данных

### Анализ производительности
- Подсчет операций сравнения и перестановок
- Вычисление эффективности алгоритмов
- Статистика производительности
- Сравнение различных подходов

## Особенности реализации

### Задание 1 - Одномерные массивы
- **Модульность:** отдельные функции для каждой операции
- **Обработка ошибок:** проверка корректности ввода
- **Дополнительная функциональность:** вычисление статистики
- **Форматированный вывод:** читаемое представление данных

### Задание 2 - Сортировка с анализом
- **Два режима работы:** ручной ввод и случайная генерация
- **Подробная статистика:** количество перестановок и эффективность
- **Оптимизация:** раннее завершение при отсутствии перестановок
- **Анализ производительности:** процент от максимально возможных перестановок

### Задание 3 - Двумерные массивы
- **Гибкость:** работа с матрицами произвольного размера
- **Генерация данных:** создание случайных матриц для тестирования
- **Наглядность:** вывод матрицы для проверки результата
- **Тестирование:** проверка на заранее известных данных

## Практические применения

Полученные навыки применимы для:
- Обработки больших объемов данных
- Анализа производительности алгоритмов
- Работы с изображениями и матрицами
- Сортировки и поиска в базах данных
- Научных вычислений и статистики
