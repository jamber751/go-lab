# Week 06, Task 03: Подсчет вхождений числа 7 в двумерном массиве

## Описание задания

Дан двухмерный массив n × m элементов. Определить, сколько раз встречается число 7 среди элементов массива.

## Требования

- Создать функцию для подсчета вхождений числа 7 в двумерном массиве
- Продемонстрировать работу с случайной матрицей
- Показать работу с тестовой матрицей с известным результатом
- Вывести матрицу на экран для наглядности

## Структура решения

### Функции

1. **`countSevens(matrix [][]int) int`**
   - Подсчитывает количество вхождений числа 7 в двумерном массиве
   - Принимает: двумерный массив целых чисел
   - Возвращает: количество найденных семерок

2. **`createRandomMatrix(n, m int) [][]int`**
   - Создает случайную матрицу размером n × m
   - Заполняет элементами от 0 до 9
   - Использует текущее время как seed для генератора случайных чисел

3. **`printMatrix(matrix [][]int)`**
   - Выводит матрицу на экран в читаемом формате

### Алгоритм

1. Проходим по всем элементам матрицы (двойной цикл)
2. Для каждого элемента проверяем, равен ли он 7
3. Если да, увеличиваем счетчик
4. Возвращаем общее количество найденных семерок

## Пример использования

```go
// Создание тестовой матрицы
testMatrix := [][]int{
    {1, 7, 3, 4, 5},
    {7, 2, 7, 8, 9},
    {0, 1, 2, 7, 4},
    {5, 6, 7, 8, 9},
}

// Подсчет семерок
count := countSevens(testMatrix)
fmt.Printf("Количество семерок: %d\n", count) // Выведет: 5
```

## Особенности реализации

- Использование срезов Go для представления двумерного массива
- Обработка матриц произвольного размера
- Демонстрация работы с генератором случайных чисел

- Тестирование на заранее известных данных

## Сложность алгоритма

- **Временная сложность**: O(n × m), где n и m - размеры матрицы
- **Пространственная сложность**: O(1) - дополнительная память не используется
