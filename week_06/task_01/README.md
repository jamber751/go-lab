# Week 06, Task 01: Работа с одномерным массивом

## Описание задания

Написать программу на Go для работы с одномерным массивом, которая выполняет следующие операции:

1. **Ввод массива**: Программа запрашивает у пользователя ввод 12 элементов действительного типа (float64)
2. **Сортировка**: Располагает элементы массива в порядке убывания
3. **Поиск экстремумов**: Находит максимальный и минимальный элементы массива
4. **Вычисление суммы**: Вычисляет сумму максимального и минимального элементов

## Требования

- Использовать тип данных `float64` для элементов массива
- Реализовать интерактивный ввод данных с проверкой корректности
- Применить алгоритм сортировки по убыванию
- Найти максимальный и минимальный элементы в исходном массиве
- Вывести все промежуточные результаты и финальный ответ

## Структура программы

### Основные функции:

- `inputArray()` - ввод массива из 12 элементов с проверкой корректности
- `printArray()` - форматированный вывод массива
- `sortDescending()` - сортировка массива по убыванию
- `findMaxMin()` - поиск максимального и минимального элементов
- `calculateAverage()` - дополнительная функция для вычисления среднего арифметического

### Алгоритм работы:

1. Пользователь вводит 12 действительных чисел
2. Программа выводит исходный массив
3. Массив сортируется по убыванию и выводится
4. Находятся максимальный и минимальный элементы в исходном массиве
5. Вычисляется и выводится их сумма


## Особенности реализации

- **Обработка ошибок**: Программа проверяет корректность ввода и предлагает повторный ввод при ошибке
- **Модульность**: Код разделен на логические функции для лучшей читаемости
- **Дополнительная функциональность**: Вычисление среднего арифметического для демонстрации работы с массивом
- **Форматированный вывод**: Числа выводятся с точностью до двух знаков после запятой

## Технические детали

- **Язык**: Go 1.21+
- **Тип данных**: `[]float64` для массива
- **Сортировка**: Используется встроенная функция `sort.Slice()` с кастомной функцией сравнения
- **Поиск экстремумов**: Линейный поиск по массиву
