# Неделя 4 - Циклы и математические ряды

## Обзор недели

Четвертая неделя посвящена изучению циклов `for` в Go и работе с математическими рядами. Студенты учатся использовать циклы для итеративных вычислений и обработки сложных математических выражений.

## Задания

### [Задание 1: Цикл for - Вычисление математического ряда](./task_01/)

**Описание:** Программа для вычисления ряда Тейлора функции arctan(X) с использованием цикла `for`.

**Ключевые темы:**
- Цикл `for` в Go
- Математические ряды и их сходимость
- Функция `math.Pow()` для возведения в степень
- Сравнение с точными значениями математических функций

**Математическая формула:**
```
X - X³/3 + X⁵/5 + ... + (-1)ᴺX^(2N+1)/(2N+1)
```

**Файлы:**
- [main.go](./task_01/main.go) - исходный код
- [README.md](./task_01/README.md) - подробное описание задания
- [go.mod](./task_01/go.mod) - модуль Go

**Запуск:**
```bash
cd week_04/task_01
go run main.go
```

### [Задание 2: Вычисление суммы и произведения конечного ряда](./task_02/)

**Описание:** Сложная программа для вычисления математического выражения с суммой и произведением, включающая обработку особых случаев.

**Ключевые темы:**
- Вложенные циклы
- Обработка особых случаев (деление на ноль, бесконечность)
- Сложные математические выражения
- Функции `math.IsInf()` и `math.IsNaN()`

**Математическая формула:**
```
W = Σ_{i=1}^{k} [(-1)^i / (i-3)^2] * Π_{n=i}^{2k} [(n^3 - 8) / (n + 2)]
```

**Файлы:**
- [main.go](./task_02/main.go) - исходный код
- [README.md](./task_02/README.md) - подробное описание задания
- [go.mod](./task_02/go.mod) - модуль Go

**Запуск:**
```bash
cd week_04/task_02
go run main.go
```

## Цели обучения

После завершения недели 4 студенты должны уметь:

1. **Использовать циклы `for`** для итеративных вычислений
2. **Работать с математическими рядами** и понимать их сходимость
3. **Обрабатывать особые случаи** в математических вычислениях
4. **Применять вложенные циклы** для сложных вычислений
5. **Использовать математические функции** для проверки результатов

## Технические навыки

- Синтаксис цикла `for` в Go
- Математические функции (`math.Pow`, `math.IsInf`, `math.IsNaN`)
- Обработка ошибок в математических вычислениях
- Работа с вещественными числами высокой точности
- Структурирование сложных алгоритмов
- Отладка математических программ

### Задание 2
