# Неделя 4, Задание 2: Вычисление суммы и произведения конечного ряда

## Описание задания

Составить схему алгоритма и написать программу вычисления суммы (произведения) конечного ряда. При разработке программы предусмотреть пропуск слагаемых (множителей), равных нулю или бесконечности.

## Математическая формула

```
W = Σ_{i=1}^{k} [(-1)^i / (i-3)^2] * Π_{n=i}^{2k} [(n^3 - 8) / (n + 2)]
```

Где:
- `k` - вводится с клавиатуры
- `Σ` - сумма от i=1 до k
- `Π` - произведение от n=i до 2k

## Требования

1. **Обработка особых случаев:**
   - Пропуск слагаемых, равных нулю
   - Пропуск слагаемых, равных бесконечности
   - Пропуск множителей, равных нулю
   - Пропуск множителей, равных бесконечности

2. **Особые точки в формуле:**
   - При `i = 3`: знаменатель `(i-3)^2 = 0` → пропуск слагаемого
   - При `n = -2`: знаменатель `(n + 2) = 0` → пропуск множителя
   - При `n = 2`: числитель `(n^3 - 8) = 0` → множитель равен нулю

## Структура программы

- `calculateW(k)` - основная функция вычисления W
- `calculateSumTerm(i)` - вычисление одного слагаемого суммы
- `calculateProduct(i, k)` - вычисление произведения для заданного i
- Обработка особых случаев с выводом информативных сообщений

1. **Проверка на деление на ноль:** Программа проверяет знаменатели перед делением
