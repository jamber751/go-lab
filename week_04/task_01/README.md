# Неделя 4, Задание 1 - Цикл for

## Описание задания

Дано вещественное число X и целое число N (> 0). Найти значение ряда:

**X - X³/3 + X⁵/5 + ... + (-1)ᴺX^(2N+1)/(2N+1)**

Этот ряд представляет собой первые N+1 членов разложения в ряд Тейлора функции arctan(X).

## Математическая формула

В общем виде ряд можно записать как:

```
∑(k=0 to N) [(-1)^k * X^(2k+1) / (2k+1)]
```

Где:
- k - индекс члена ряда (от 0 до N)
- (-1)^k - знак члена (чередуется)
- X^(2k+1) - степень X
- (2k+1) - знаменатель

## Требования

1. Использовать цикл `for` для вычисления суммы ряда
2. Обработать ввод пользователя с проверкой корректности
3. Вывести промежуточные вычисления для наглядности
4. Сравнить результат с точным значением arctan(X)

## Структура программы

- `calculateSeries(x, n)` - функция для вычисления ряда с использованием цикла for
- `main()` - основная функция с вводом данных и выводом результатов

## Особенности реализации

1. **Цикл for**: Используется для итерации от k=0 до k=N
2. **Математические функции**: `math.Pow()` для возведения в степень
3. **Проверка ввода**: Валидация значения N
4. **Наглядность**: Вывод каждого члена ряда
5. **Сравнение**: Сравнение с точным значением arctan
