# Week 14, Task 01: Менеджер заметок

## Описание
CLI-приложение для управления заметками с сохранением в JSON.

## Использование
```bash
# Добавить заметку
go run main.go -add "Текст заметки"

# Показать все заметки
go run main.go -list

# Удалить заметку
go run main.go -delete 1

# Поиск
go run main.go -search "текст"
```
