# Неделя 14 - Финальный проект

## Задание: CLI-приложение "Менеджер заметок"

### [task_01: Notes Manager](./task_01/)

Консольное приложение для управления заметками с сохранением в JSON.

**Функции:**
- Добавление заметок
- Просмотр списка
- Удаление по ID
- Поиск по тексту
- Сохранение/загрузка из файла

**Темы:**
- Структуры и методы
- Работа с файлами (JSON)
- Флаги командной строки
- Обработка ошибок
