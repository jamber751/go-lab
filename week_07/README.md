# Неделя 7 - Структуры и интерфейсы

## Задания

### [Задание 1: Структура "Книга"](./task_01/)
Создать структуру Book с полями: название, автор, год, цена. Реализовать методы вывода и расчёта скидки.

### [Задание 2: Интерфейс "Фигура"](./task_02/)
Создать интерфейс Shape с методами Area() и Perimeter(). Реализовать для Circle и Rectangle.

### [Задание 3: Встраивание структур](./task_03/)
Создать структуру Person и Employee с встраиванием. Реализовать методы для обоих типов.
