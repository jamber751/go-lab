## Как использовать

1. Перейдите в папку нужной недели
2. Выберите задание
3. Изучите README.md для понимания задачи
4. Запустите код: `go run main.go`

## Требования

- Go 1.19 или выше
- Git для версионирования

## Правила работы

- Каждое задание должно быть в отдельной папке
- Код должен быть отформатирован с помощью `gofmt`
- Обязательно комментировать код на русском языке
- Каждое задание должно иметь README.md с описанием
